Sales Quotation Split

Divide large quotation into sub quotations

Allow Quotation Splitting in Sales settings

By installing this module, you will find option under Settings >> Configuration >> Sales menu.

Split Quotation Button in Quotation Form

Once you tick the option "Allow Quotation Splitting" under Settings >> Configuration >> Sales menu, a button named as "Split Quotation" will be visible in any Quotation form. You can split the quote only in case of when it is in "Draft" state and having more then one sale order lines.

Select order lines and move them to new Quotation

By clicking on "Split Quotation" button in any Quotation form, you will able to select Order lines and move them to new Quotation. Here, you can change customer and pricelist also. So in new quotation, you will have changed price according to selected pricelist.

Naming pattern of Splitted Quotations

Once you split any quotation from main quotation, you will have that main Quotation reference no in splitted quotation. For example, if you split SO015 to new quotation then reference no of sub quote will be S0015/1 and if you further devide SO015 to again in any sub part then its reference no would be SO015/2 despite of even S0015/1 is deleted.

Disable splitted quote naming pattern

You can disable naming pattern from splitted quotation by company wise. In order to do that go to Settings >> Companies >> Companies and open any company and uncheck the option "Keep Split Quotation Name Pattern?". However by disabling the naming pattern in splitted quotation, you will still able to see history in any quotation, from "Splitted From" field.
